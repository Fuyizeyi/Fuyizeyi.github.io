GitHub注册教程

（1）进入创建仓库页面

（2）创建仓库

Owner就是仓库的持有者（就是你）
仓库名推荐起： xxxx.github.io
下面的仓库类型选择Public即可
以及最后的Add a README file即可
最后Creat repository就创建成功啦~~

（3）看看你的仓库吧

如果没有自动跳转，那么你回到首页，你就能在最左边找到你刚才创建的仓库名，点进去就是啦

2.本地ssh链接仓库

仓库创建后
在电脑上随便找个文件夹，进入后右键打开git bash
或者win+R打开运行，输入cmd打开命令窗口

总之你得打开一个终端
使用命令 ssh 生成本地ssh密钥，这条命令将会以电脑现在登录的用户生成密钥文件

打开本地生成的ssh文件
C:\Users\你的用户名.ssh
打开id_rsa.pub后复制密钥代码，看清楚了，尾缀是pub，千万别打开错了，打开后ctrl+A然后ctrl+C复制文档内所有内容

粘贴到GitHub的ssh key内
回来GitHub进入点击头像进入setting

在左侧找到SSH and GPG keys点击

在SSH keys的右边有个new SSH key，点击进入创建密钥

title是密钥的名字，用于自己识别，随便起
key为我们刚才在win10本地使用指令生成的id_rsa.pub内复制的内容，全选复制后粘贴到key这一框即可（刚才复制了可以直接粘贴）
最后直接点击Add SSH key即可

ssh密钥链接本地就创建成功了

测试ssh是否绑定成功：

如果返回successfully就是成功了
