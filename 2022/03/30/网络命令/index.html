一、write

1、基础信息

指令名称: write
指令所在路径:/usr/bin/write
执行权限:所有用户（仅可给在线的用户发送）
语法: write <用户名>
功能描述:给用户发信息，以Ctrl+D保存结束，ctrl+back删除

2、使用

  • 给用户usertest发送信息”出来吃饭啊“

二、wall

1、基础信息

指令名称: wall
命令英文原意: write all
指令所在路径:/usr/bin/wall
执行权限:所有用户
语法: wall [message]
功能描述:发广播信息

四、ipconfig（现在centos 7没用了，用 ip addr ）

1、基础信息

命令名称:ifconfig
命令英文原意: interface configure

执行权限: root
语法: ifconfig 网卡名称IP地址
功能描述:查看和设置网卡信息

五、mail

1、基础信息

命令名称: mail
命令所在路径:/bin/mail
执行权限:所有用户
语法: mail [用户名]
功能描述:查看发送电子邮件

2、使用

  • 给root发一封邮件

  • root对邮件进行接受
    上线并不会直接显示有邮件，输入mail显示有邮件，前面有序列号，如图中为1
    在下面敲入1快速查看邮件
  • 删除邮件
    在&后面写 d +序列号 相应删除
