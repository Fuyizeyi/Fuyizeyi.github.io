其他权限命令

一、chown

1、基础信息

命令名称：chown
命令英文原意：change file ownership
命令所在路径：/bin/chown
执行权限：所有用户
语法：sschown [用户] [文件或目录]
功能描述：改变文件或目录的所有者

2、使用

※只有文件所有者和root可以改变文件的属性
只有root可以改文件所有者

  • 改变文件的拥有者（使用root账户）

二、chgrp

1、基础信息

命令名称：chgrp
命令英文原意：change file group
命令所在路径：/bin/chgrp
执行权限：所有用户
语法：chgrp [用户组] [文件或目录]
功能描述：改变文件或目录的新属组

2、使用

  • 改变文件所属组（把123这个文件给testgroup这个组）

三、umask

1、基础信息

命令名称：umask
命令英文原意：the user file-creation mask
命令所在路径：Shell内置命令
执行权限：所有用户
语法：

功能描述：显示、设置文件的缺省权限
※这里缺省=默认=default
※默认创建的文件夹会遵循默认的权限，而创建的文件，他会自动将所有的用户组的x即执行权限去除，实际上缺省权限还是一样的（安全性考虑，病毒不能执行就不能运作了）

2、使用

umask在数字表达中，即直接键入umask不适用-S命令，它会显示数字，他的数字显示的是缺少的权限，就是实际上是r-x数字为5，而umask现实的是-w-为2，但是你用-S用rwx模式显示他的文件的时候，他还是正常显示的是r-x
